Magento 2.2.x 技术栈要求

操作系统(Linux x86-64)

Linux 分时系统，如 RedHat Enterprise Linux (RHEL), CentOS, Ubuntu, Debian等等.

内存要求

升级Magento和你从Magento Marketplaces和其他源上获取的扩展可能要求你的系统至少2G的内存，如果你使用的系统内存少于2G，我们建议您创建一个交换文件; 否则, 你的升级可能会失败.

Composer (最新的稳定版)

Composer贡献Magento2基础代码和扩展的开发者将会用到

Web服务器

  • Apache 2.2 or 2.4

    In addition, 你必须启用Apache的mod_rewrite和mod_version模块. mod_rewrite模块允许服务器进行路由重写。 mod_version模块为不同的httpd版本提供了一个灵活的版本检查.更多信息请参考我们的Apache文档.

  • nginx 1.x

数据库

MySQL 5.6, 5.7

Magento同样支持MySQL NDB Cluster 7.4.*, MariaDB 10.0, 10.1, 10.2, Percona 5.7以及其他的基于MySQL兼容的技术的数据库.

Magento仅支持MariaDB使用和MySQL兼容的特征，MariaDB并不完全兼容MySQL的所有特征，所以将你应用一个新的特征到Magento模块时，你应当先研究一下它们的兼容性,

PHP

7.0.0, 7.0.1 7.0.2 7.0.3 7.0.4 7.0.5 7.0.6–7.0.x 7.1.x

所需的PHP扩展

CentOS和Ubuntu下php安装命令包含了安装他们扩展的步骤

  • bc-math (Magento Commerce 仅2.2.0 - 2.2.3. Magento Commerce及Magento Open Source2.2.4以下.)
  • ctype
  • curl
  • dom
  • gd, ImageMagick 6.3.7(或更新版本)或两个都装
  • intl
  • mbstring
  • mcrypt
  • hash
  • openssl
  • PDO/MySQL
  • SimpleXML
  • soap
  • spl
  • libxml
  • xsl
  • zip
  • json
  • iconv

PHP OPcache

基于性能的原因，我们强烈建议你检查你的PHP OPcache是否开启. 在多数PHP中OPcache都是开启的. 要验证你是否已经正确安装, 请参考我们的CentOS或Ubuntu的PHP文档.

如果你要单独地安装OPcache, 请参考PHP OPcache安装文档.

PHP配置

我们建议您修改PHP的一些配置, 诸如 memory_limit,这样可以避免在使用Magento的过程中的一些常见的问题

了解更多, 请参考PHP配置要求.

SSL

  • 要使用Https你需要一个有效的安全证书
  • 自己签发的证书是不支持的

  • 传输层安全协议(TLS)要求 - PayPal和repo.magento.com都要求使用TLS 1.1或更新的版本:

    • 关于PayPal的更多信息

    • repo.magento.com的更多信息

邮件服务server

邮件收发代理(MTA) 或一个简单邮件服务(SMTP)

Magento能使用以下技术:

  • 页面缓存及Session存储Redis版本3.2 (兼容2.4+ )
  • Varnish版本4.x或5.0
  • 用于session存储的memcached(最新的稳定版)，您可以选择使用PHP的memcache或memcached扩展(最新的稳定版)连接它们

仅Magento Commerce支持

  • Elasticsearch

    Magento Commerce(版本2.2.x)支持以下的Elasticsearch版本:

    • Elasticsearch 5.x
    • Elasticsearch 2.x

    Magento 2.2.3 使用Elasticsearch PHP客户端(版本5.1).在2.2.3以前使用的PHP客户端是2.0.

  • RabbitMQ 3.5.x (兼容2.0及其以上)

    RabbitMQ用于发布消息到队列，供用户订阅和导步地接收.

  • 三个主数据库

    这些主数据库为Magento2应用程序的不同功能区提供了可伸缩的优势, 诸如checkout, 下单, 以及Magento2应用程序的所有数据表.
