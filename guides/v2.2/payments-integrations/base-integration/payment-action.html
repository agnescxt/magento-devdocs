添加支付动作

For each payment action available for the payment method, you must implement the following:

  • Creating a request with payment details. Described in 从前台获取支付信息到后台.
  • Request processing using response handler and response validator.
  • Specify and configure the gateway command. Described in the 网关命令 topic.
  • Add the command to the commands pool, as described in Command Pool.

Configure the command

The gateway command for the payment action must be configured in the di.xml file of your module. Conventionally, its location must be <your_module_dir>/etc/di.xml

Configure the command as described in 网关命令.
