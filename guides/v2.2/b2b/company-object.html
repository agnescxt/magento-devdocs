服务名称

companyCompanyRepositoryV1

REST接口

CompanyInterface 参数

下面的表格列出了CompanyInterface的参数定义

</tr> </table> ### 创建一个公司 以下是创建公司并分配默认共享产品目录(`customer_group_id`)的一个例子。此公司管理员(`super_user_id`)必须是先前定义的一个`customer_id` **样例用法** `POST /V1/company/` **载荷** **响应** ### 更新公司 下面的调用改变了公司的状态到拒绝状态(`2`),并解释了原因 **样例用法** `PUT /V1/company/2` **载荷** **响应** ### 返回公司的所有信息 此调用返回了指定公司的详细信息 **样例用法** `GET /V1/company/2` **载荷** 无 **响应** ### 删除公司 当你删除一个公司，Magento分配"不活动"状态给所有的公司成员。系统也将从所有公司成员的客户信息中移除公司ID **样例用法** `DELETE /V1/company/2` **载荷** 无 **响应** `true`,指示请求成功 ### 搜索公司 下面的调用返回所有位于加利福尼亚的公司(`region_id` = `12`) 参考[使用REST API搜索](/guides/v2.2/rest/performing-searches.html)了解更多关于构造查询的信息 **样例用法** `GET /V1/company?searchCriteria[filter_groups][0][filters][0][field]=region_id&searchCriteria[filter_groups][0][filters][0][value]=12&searchCriteria[filter_groups][0][filters][0][condition_type]=eq` **载荷** 无 **响应**

名称描述格式要求
id系统生成的公司ID integer 更新和删除时需要
status0 - 待审核
1 - 已审核
2 - 已拒绝
3 - Blocked 		integer 可选
company_name 公司名称string 创建和更新公司时需要
legal_name 法人名字 string 可选
company_email 公司官方邮箱地址，不一定要唯一string更新和创建公司时需要
vat_tax_id 公司的增值税号string 可选
reseller_id 公司分销商的唯一IDstring 可选
comment 公司的更多细节信息string可选
street公司注册的街道地址，可以包含一行或两行。/td>Array[string]更新和创建公司时需要
city公司所在城市string 更新和创建公司时需要
country_id公司注册的国家string 创建和更新公司时需要
region州或省string更新和创建公司时需要
region_id分配给州或省的IDstring 可选
postcode公司的邮政编码string 更新和创建公司时需要
telephone公司的联系电话string更新和创建公司时需要
customer_group_id 定义公司的共享产品目录。值`1`分配给默认共享产品目录integer更新和创建公司时需要
sales_representative_id公司销售代表的用户IDinteger可选
reject_reason指定为什么公司请求成为B2B客户被拒string可选
rejected_at指示公司被拒的时间截string可选
super_user_id公司管理员的`customer_id`，当创建一个公司时，`customer_id`必须已经存在integer更新和创建公司时需要
