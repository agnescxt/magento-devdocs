支持的版本

Migration from versions

The 数据迁移工具 supports migration from the following versions:

  • Magento Commerce:

    • 1.11.x

    • 1.12.x

    • 1.13.x

    • 1.14.x

  • Magento Open Source:

    • 1.6.x

    • 1.7.x

    • 1.8.x

    • 1.9.x

If you migrate from Magento Open Source to Magento Commerce, the following versions are supported:

  • 1.6.x

  • 1.7.x

  • 1.8.x

  • 1.9.x

Migration to version

To know the supported Magento 2 version (the one you’re migrating to), see the 数据迁移工具’s Release page.
